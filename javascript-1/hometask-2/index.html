Задача №1

Дан код:
var a = 1, b = 1, c, d;
c = ++a; alert(c); // 2 // Ответ: Эта операция выполняет инкремент (прибавляет 1) операнда и возвращает его значение. Если используется префикс, возвращается значение, полученное после выполнения инкремента.
d = b++; alert(d); // 1 // Ответ: Если это постфикс, возвращается значение, которое было до выполнения инкремента.
c = (2+ ++a); alert(c); // 5 // Ответ: Сначала выполняется инкремент префикс, увеличение a=a+1, потом другие операции. Т.е. 2 + ((a=2)+1) присвоить c.
d = (2+ b++); alert(d); // 4 // Ответ: Сначала выполняется операция сложения 2 + 2 = 4 присвоить c, а потом уже инкремент постфикс b=b+1.
alert(a); // 3 // Ответ: Значение "a" увеличивалось в предыдущих операциях инкремента 2 раза на 1, т.е. a=a+1+1
alert(b); // 3 // Ответ: Значение "b" увеличивалось в предыдущих операциях инкремента 2 раза на 1, т.е. b=b+1+1
Почему код даёт именно такие результаты?
